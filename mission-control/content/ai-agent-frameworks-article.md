## 1. LangChain / LangGraph

**Overview:** Leading open-source framework with extensive tooling for building LLM-powered applications.

**Key Features:**
- Plan-and-execute, Multi-agent, ReAct, Self-ask architectures
- Human-in-the-loop approval workflows
- Native streaming support
- LangSmith for observability & debugging
- Extensive off-the-shelf tool library

**Pricing:** 
- LangChain: Free (open-source)
- LangSmith: Free tier + paid plans (~$40/mo for pro)

**Best For:** Developers wanting full control & customization

---

## 2. CrewAI

**Overview:** Enterprise-focused platform for orchestrating autonomous AI agents into collaborative teams.

**Key Features:**
- Visual editor + AI copilot for non-coders
- Pre-built integrations (Gmail, Slack, Salesforce, HubSpot)
- Real-time workflow tracing
- Agent training & task guardrails
- Serverless scaling

**Pricing:**
- Open-source: Free
- AMP Cloud: Custom pricing

**Best For:** Enterprises needing scalable multi-agent orchestration

---

## 3. AutoGen (Microsoft)

**Overview:** Microsoft's open-source framework for creating multi-agent AI applications.

**Key Features:**
- Multi-agent orchestration
- AutoGen Studio (no-code GUI)
- AutoGen Bench for evaluation
- Cross-language support (Python, .NET)
- Magentic-One pre-built multi-agent team

**Pricing:** Free (MIT License)

**Best For:** Enterprise/Microsoft shops needing reliability

---

## 4. LlamaIndex

**Overview:** Developer-first framework specializing in document-centric agents and RAG.

**Key Features:**
- LlamaParse: Industry-best document parsing (90+ file types)
- Schema-based extraction with confidence scores
- Event-driven async workflows
- Enterprise RAG optimization

**Pricing:**
- LlamaCloud Free: 10,000 credits/month (~1000 pages)
- Open-source: Free

**Best For:** Document automation, enterprise knowledge bases

---

## 5. AutoGPT

**Overview:** Autonomous AI agent platform for visual/no-code workflow automation.

**Key Features:**
- Low-code visual agent builder
- Workflow management with connected blocks
- Pre-built agent library
- Continuous 24/7 operation

**Pricing:** Free (self-host) | Cloud: Coming soon

**Best For:** Non-developers wanting visual automation
